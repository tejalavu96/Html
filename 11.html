HTML to Contact Form 7 for WordPress

Fast tool to convert your HTML in code Contact Form 7 (plugin) for generate contact forms in WordPress.


V1.0

  • Compatibility with HTML5 code that contains "" or ''.
  • Support for inputs of type text, email, url, number.
  • Support for textarea.
  • Attributes availables: class, id, name, value, placeholder.
